# Ani47
CLI app to browse and watch on Anime47
# Installation
```
$ git clone https://github.com/CleveTok3125/Ani47/
$ cd Ani47
$ pip install -r requirements.txt
```
# Usage
Default:
```
$ python main.py
```
Quick Search:
```
$ python main.py <anime_name>
```
View last watched anime:
```
$ python main.py !last
```
